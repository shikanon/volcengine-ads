# 广告视频打分 Prompt

> 核心约定:
> - 用 **doubao-seed-2.0 pro 直接输入视频**(不抽帧)完成理解与打分。
> - **每类广告有自己的一套维度**,打分结果**不做跨类统一**,只展示该类型的维度。
> - **对比只在同类型广告之间有意义**(品牌 vs 品牌、买量 vs 买量、创意 vs 创意)。
> - 每条素材最后由大模型给出**一段针对性的分析与优化建议**。

---

## 一、整体流程

```
视频  ──►  ① 选择分类  ──►  判定类型
                                  │
              ┌───────────────────┼───────────────────┐
              ▼                   ▼                   ▼
        品牌打分Prompt        买量打分Prompt        创意打分Prompt
       (品牌专属维度)        (买量专属维度)        (创意专属维度)
              │                   │                   │
              └───────────────────┴───────────────────┘
                                  ▼
              该类型的维度分 + 一段针对性分析与优化建议
                                  ▼
                  客户端按该类型的维度渲染(雷达图等)
```

- **不输出跨类可比的总分**。每类各自一套维度,维度含义与数量都可不同。


每个维度 0–100 分,由大模型直接给出。**不再做跨类归一,也不合成统一总分**——因为不同类型的维度本就不可比。

## 二、分类 Prompt

```
你是广告视频分类专家。观看完整视频,判断类别并给出三类概率(和为1)。
- 品牌广告:心智渗透与品牌好感。高频Logo/Slogan/品牌色、情感叙事、TVC质感、CTA弱、不强调价格。
- 买量广告:直接促转化。强CTA、价格/优惠、痛点开场、3秒钩子→信任→诱惑→行动结构、引导落地页。
- 创意广告:制造记忆点与自传播。反转/悬念/梗、视觉双关、UGC风格、强卡点、模板化弱。
输出 JSON:{ "category":"...", "category_probs":{...}, "reason":"引用关键画面/台词与时间点" }
```

## 三、品牌广告 打分 Prompt

> 融入「品牌一致性+ 制作质量+ CAP 品牌辨识度/遮Logo测试」。

```
你是品牌广告评估专家。先过合规红线;通过后观看完整视频(这是一条品牌广告,目标=心智渗透与品牌好感)。
对以下维度各打0-100分,给出证据时间点。最后给一段针对该素材的整体分析与优化建议。

【维度与考察要点(品牌广告)】
1. 品牌露出与一致性:Logo/Slogan/品牌色的频次·时长·显著度;是否符合品牌规范(Brand Safety);品牌与内容是否强绑定。
   ↳ 辅助判断:做"遮Logo测试"——遮住Logo后是否仍能认出该品牌,能则识别度高。
2. 制作精良度(声音+画面):音质清晰无杂音/音量适中;画质清晰稳定、色彩光影舒适、构图运镜有质感、无马赛克遮挡;声画同步、字幕匹配、剪辑流畅。
3. 情感叙事与好感度:是否通过故事/情绪/价值观引发共鸣,提升品牌好感(可参考情绪共鸣度)。
4. 黄金3秒吸引力:开场是否留人(品牌广告允许情绪化慢开场,但需有美感或悬念)。
5. 新颖度/差异化:是否区别于同品类品牌广告,有无记忆点(可参考创意跳跃度)。

【评分倾向】品牌露出与一致性 > 制作精良度 > 情感叙事 > 黄金3秒 ≈ 新颖度。CTA不作为扣分项。

输出 JSON:
{ "category":"品牌广告","compliance_pass":true,
  "dimension_scores":{"品牌露出与一致性":0,"制作精良度":0,"情感叙事与好感度":0,"黄金3秒吸引力":0,"新颖度差异化":0},
  "evidence":{"维度名":"时间点+依据"},
  "analysis":"整体分析(优点+短板+原因,品牌资产视角)",
  "suggestions":["可执行优化建议1","建议2"] }
```

## 四、买量广告 打分 Prompt

> 融入「爆款结构:3秒钩子→内容信任力→引导诱惑力→转化行动力」+「广告心理:损失厌恶/挑战欲/权威背书」+「激励素材:信息清晰度/参与路径」。

```
你是买量广告(效果广告)评估专家。先过合规红线;通过后观看完整视频(目标=促成下载/购买转化)。
对以下维度各打0-100分,给出证据时间点。最后给一段针对该素材的整体分析与优化建议。

【维度与考察要点(买量广告,对齐爆款四段结构)】
1. 黄金3秒钩子:开场是否用痛点/冲突/利益点/悬念强钩子留人(可调用损失厌恶"错过即亏"、挑战欲"只有1%的人能…"等心理触发)。
2. 内容信任力:是否有权威背书/真实演示/数据/品牌露出,建立可信度(避免空口承诺)。
3. 利益点与诱惑力:卖点/优惠/价格/福利是否说清、量化、有说服力;是否制造紧迫感(限时/限量/稀缺)。
4. CTA与转化行动力:行动号召是否明确、时机恰当;参与路径/下一步是否清晰易懂(参考"参与路径易懂")。
5. 转化链路流畅度:吸引→信任→诱惑→行动 是否顺畅无断点。
6. 制作合格度:画质清晰流畅、口播清晰、字幕匹配(买量不追求电影感,但不能廉价/中断)。

【评分倾向】黄金3秒钩子 ≈ CTA行动力 > 利益点诱惑力 > 内容信任力 ≈ 转化链路 > 制作合格度。
【红线提醒】夸大宣传/虚假利益点直接计入合规红线。

输出 JSON:
{ "category":"买量广告","compliance_pass":true,
  "dimension_scores":{"黄金3秒钩子":0,"内容信任力":0,"利益点与诱惑力":0,"CTA与转化行动力":0,"转化链路流畅度":0,"制作合格度":0},
  "evidence":{"维度名":"时间点+依据"},
  "analysis":"整体分析(转化链路视角:哪一段是瓶颈)",
  "suggestions":["可执行优化建议1","建议2"] }
```

## 五、创意广告 打分 Prompt

> 融入「CAP:创意跳跃度/视觉冲击力 + 去文案测试/复述测试」+「用户研究:剧情紧凑度/情绪共鸣/结尾引导力」+「游戏:神秘感/未完待续悬念」。

```
你是创意广告评估专家。先过合规红线;通过后观看完整视频(目标=记忆点与自传播)。
对以下维度各打0-100分,给出证据时间点。最后给一段针对该素材的整体分析与优化建议。

【维度与考察要点(创意广告)】
1. 新颖度/创意跳跃度:创意是否原创、打破套路、有"没见过"的惊喜(隐喻/视觉双关/语义反转/反差);模板化跟风则大幅扣分。
2. 黄金3秒吸引力:开场是否用反转/悬念/反差/梗/未完待续 迅速抓住注意力。
3. 节奏与紧凑度:剪辑节奏、音乐卡点、信息密度是否带来观看快感;有无冗余(参考"每10-15秒一个转折")。
4. 记忆点/传播性:是否有可二创/可讨论/可模仿的钩子。
   ↳ 辅助判断:做"复述测试"——看完后用户能否在24小时内复述这条创意,能则记忆点强。
5. 情绪共鸣度:是否呈现易代入的情感冲突或生活痛点,引发共情。
6. 信息传达清晰度:好玩的同时是否让人记住要传达什么(避免"只搞笑没记住产品")。
   ↳ 辅助判断:做"去文案测试"——删掉字幕后画面是否仍能传达核心信息。

【评分倾向】新颖度/创意跳跃度 > 黄金3秒 > 节奏紧凑 ≈ 记忆点 > 情绪共鸣 > 信息传达。硬CTA与品牌露出不作为主要评分项。

输出 JSON:
{ "category":"创意广告","compliance_pass":true,
  "dimension_scores":{"新颖度创意跳跃度":0,"黄金3秒吸引力":0,"节奏与紧凑度":0,"记忆点传播性":0,"情绪共鸣度":0,"信息传达清晰度":0},
  "evidence":{"维度名":"时间点+依据"},
  "analysis":"整体分析(创意与传播视角)",
  "suggestions":["可执行优化建议1","建议2"] }
```

五、客户端可视化

### 5.1 展示组件

**① 维度雷达图(主视觉)**
- 按返回的 `dimension_scores` **动态渲染轴**(不同类型轴数量/名称不同)。
- 不展示跨类总分;若需对比,只叠加**同类型素材**的维度作为参考多边形。

**② 维度条形图(辅助)**
- 每维度一条横向进度条 + 分值,按高低排序,精确读数、快速定位短板。

**③ 分析与建议卡片(最终落点)**
- 展示 `analysis`(一段针对性分析)+ `suggestions`(优化建议列表)。
- `evidence` 的时间点可点击跳转视频对应秒。

### 5.2 同类型对比(唯一的对比方式)

- 对比入口只允许选择**同一类型**的素材集。
- 雷达图叠加多条同类素材,或条形图分组对比同一维度。
- **禁止跨类型对比**(品牌的"品牌露出"与买量的"CTA清晰度"无可比性)。
